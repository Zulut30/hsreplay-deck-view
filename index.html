Все варианты карточек для Hearthstone-страниц

Одна маленькая библиотека без сборки: подключил CSS/JS, передал готовые карты или card id, получил HSReplay-похожие плитки, иконки и архетипы.

Deck tiles mana + rarity + art
Circle strip compact deck row
Square strip arena-style icons
Archetype row layered background arts
Stone portrait battlegrounds frame
Synergy card combo chain
Variant 01

Плитки карт

Классический список: стоимость слева, цвет редкости, затемнение под названием, счетчик копий или звезда для одиночной легендарки.

Variant 02

Круглые иконки

Узкая строка для карточки колоды: арты в кружках, `×2` для копий и `★` для легендарок.

Variant 03

Квадратные иконки

Более плотный вариант для длинных рядов: ровные квадратные превью, мягкий радиус и такие же бейджи количества.

Variant 04

Карточки архетипов

Несколько артов берутся как фоновые панели, накладываются через диагональный разделитель и остаются достаточно мягкими, чтобы название читалось сразу.

Variant 05

Каменный круг

Большой Battlegrounds-портрет: арт карты из `256x`, круглая маска и каменная рамка `minion-frame` поверх изображения.

Dragon cycle

Крупный портрет для композиции, героя или ключевой карты.

Variant 06

Карточка синергии

Несколько карт или миньонов в цепочке с `+` или стрелкой: удобно показывать комбо, муллиган-план, BG-связки и итоговый payoff.

Reference

Редкости и стоимость

Быстрая проверка цветов, длинных названий и диапазона маны от 0 до 10.

All rarities

Costs 0-10

Drop-in API

Подключение

Сайт на GitHub Pages использует те же публичные методы, что описаны в README. 

<link rel="stylesheet" href="src/hsreplay-deck-view.css">
<script src="src/hsreplay-deck-view.js"></script>
<script>
  HSReplayDeckView.renderDeck("#deck", cards);
  HSReplayDeckView.renderIcons("#icons", cards);
  HSReplayDeckView.renderSquareIcons("#squares", cards);
  HSReplayDeckView.renderArchetypes("#archetypes", archetypes);
  HSReplayDeckView.renderStonePortraits("#stone", portraits);
  HSReplayDeckView.renderSynergies("#synergies", synergies);
</script>
